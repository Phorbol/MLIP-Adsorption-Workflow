# Examples Index

这个目录现在有两份 notebook，定位不同：

- `progressive_teaching_workflow.ipynb`
  - 面向第一次上手这个 repo 的用户。
  - 按“最小高层 API -> 正式 MACE 生产用法 -> 困难异质支撑 case”渐进展开。
  - 更偏教学叙述，强调为什么这样配、应该看哪些输出。

- `full_usage.ipynb`
  - 面向已经熟悉 repo 结构、想快速浏览更低层能力的用户。
  - 更接近全功能示例，把 surface/site/pose/basin/node/conformer-md 串起来展示。
  - 适合作为 API 参考和功能覆盖总览，不是最佳入门教程。

- `FULL_USAGE.md`
  - 面向命令行用户的文字版入口索引。
  - 更适合配合 `tools/` 里的脚本一起看。

如果你是第一次打开这个项目，先从 `progressive_teaching_workflow.ipynb` 开始。
